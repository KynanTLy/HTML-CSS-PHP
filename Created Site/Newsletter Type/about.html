About the Site

The goal of this site is to collect all the postive impact different video games had on a person's life. A secondary purpose is to show other who are experincing a low time in their life can find inspiration, motivation, relate in other video games that had benefitted others. Therefore not only serving as a source where we can map and record all the postive of different video games, but also provide a resource that allows to discover other video games as well.

Therefore this site would be an exploratory site to try and learn different ways to accomplish this goal

Special Features

Allow for the user to add their own rating to a game that then will be displayed on the home page for all to see. This will be accomplished via some form of data visualization from data from the SQL databased
